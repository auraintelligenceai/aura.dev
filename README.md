# 🎮 Pixel Portfolio

A retro-futuristic, pixel-perfect portfolio built with **React + TypeScript + Vite + Tailwind CSS**.

![Portfolio Preview](./screenshot.png)

---

## 🚀 ONE-CLICK DEPLOY

### Option 1: Vercel (Recommended)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

**Or run locally:**
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
cd portfolio-pixel
vercel --prod
```

---

### Option 2: Netlify Drop (Easiest)

1. Go to **[app.netlify.com/drop](https://app.netlify.com/drop)**
2. Drag the `dist/` folder from this project
3. Get instant live URL! 🎉

---

### Option 3: GitHub Pages

```bash
# Build first
npm run build

# Push dist folder to GitHub
git add dist -fgit commit -m "Deploy to GitHub Pages"
git subtree push --prefix dist origin gh-pages
```

---

### Option 4: Local Preview

```bash
npm run preview
```

Then open: `http://localhost:4173`
